# kubrick

Simple movie database with simple websocket experiments.

## Usage

First build clojurescript
```
lein cljsbuild once
```

Start repl of your choice, e.g. in emacs open server.clj and run within eshell :

```
lein repl
(run)
(run-ws)
```
